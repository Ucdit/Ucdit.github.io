北漂生活一

发表于 2016-07-23   |   分类于 北漂生活   |     |    阅读次数

我是2015年本科毕业，到现在整整工作一年，刚毕业在上海一家金融互联网公司做前端开发，后面因为一些原因，半年之后，我到了北京工作，收到了美团的offer。
我刚到北京的第一感觉是旧！刚到北京，在动车上，就感觉北京的树都是光秃秃的，到处都是一片黄，很美生机的一座古城。北京市一座古老的城市，历史悠久，交通方面比其他城市早发达很多年，设备建设也比较早，所以给人很旧的感觉，比如说地铁站。北京的地铁站跟上海的地铁站比起来，上海的还是干净很多的，最重要的一点是，上海的地铁站，每个地铁开门的地方，抬头就能看到站牌信息（地铁行驶方向以及站点信息），这也是我刚到北京不习惯的地方，北京的地铁站牌信息要离好远才有一个，不过北京地铁站服务人员很多。说到服务方面，还有一点很纳闷的是，北京乘地铁的人很多，过安检都要排队很长，其实每个门口都有两台安检机子，为什么西二旗地铁站要空着一台机器不用，让大家挤着一个地方过呢？这是我很搞不懂的！
北京第二感觉，其实雾霾没有那么严重，可能是我来的时候是3月中旬，那时候雾霾没那么严重，夏天空气其实挺好的，有蓝天，下雨不多，只是偶尔有点雾霾，雾霾严重天，应该在冬天比较多，特别是供暖气的时候。
北京第三感觉，就是堵，北京车超级多，上班路上，五环外，我住在西二旗附近，早上上班，走路都比开车快！so，后面我们就买了电动车啦！
北京第四感觉，发家致富机会多，各种uber，滴滴司机，外卖配送，地摊烤串等，即使没高文凭，也有很多就业机会。
