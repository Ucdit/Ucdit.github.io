三、购买域名

推荐godaddy或者阿里的万网这两个比较靠谱，不过godaddy的比较便宜。

  • 在搜索栏里面输入自己想购买的域名，搜索，如果域名已经被购买，请换个域名重新搜索。
  • 域名添加到购物车，付款即可。
  • 付款成功，可以在【我的产品】下，看到域名的项，点击管理，看到已经购买的域名
  • 域名未激活，激活方式：邮件认证之后，域名激活成功。
  • 设置DNS，用DNSpod，快，免费，稳定。
  • 添加域名之后，DNSpod会自动导入设置内容，但是要修改goddaddy修改DNS地址

四、Godaddy修改DNS地址（坑）

这点是很多新人不知道的而且很重要的一步。

  • 进入【我的产品】点击域名行的【管理】
  • 点击域名，进入dns管理界面
  • 域名服务器管理
  • 将godaddy的Nameservers更改成f1g1ns1.dnspod.net和f1g1ns2.dnspod.net
  • dns地址修改生效时间比较长大约72小时之内

五、github上建立仓库

  • 仓库名字例如：ucdit.github.io
  • 选择public
  • 点击【create repository】完成

六、关联仓库到blog

  • 进入blog的目录，找到source目录，在source目录下创建CNAME文件
  • 写入买的域名，例如：learnfe.info，保存
  • 在blog下_config.yml添加（deploy:type: git repo: https://github.com/Ucdit/Ucdit.github.io.git）
  • hexo g、hexo s、hexo deploy（编译部署上线）
  • 浏览器输入learnfe.info就可以看到默认blog首页
